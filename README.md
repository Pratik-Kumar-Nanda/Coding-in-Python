# Python String Methods Playground

A beginner-friendly Python project that demonstrates useful string methods and operations.

## Features

- Basic string methods
- Search methods
- Replace methods
- Split and strip examples
- User input examples
- Reverse string example

## Methods Covered

- len()
- capitalize()
- upper()
- lower()
- title()
- startswith()
- endswith()
- count()
- find()
- replace()
- split()
- strip()

# Python Lists and Tuples Practice

A beginner-friendly Python practice file containing examples of **lists, tuples, tuple unpacking, palindrome checking, sorting, and counting operations**.

## 📌 Overview

This project contains simple Python exercises focused on learning:

- List creation and manipulation
- Taking user input
- Palindrome checking using lists
- Counting elements in a list
- Sorting lists
- Tuple unpacking with `*` operator
